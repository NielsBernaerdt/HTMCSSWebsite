UnrealFormations

C++ based UE4 project

Description:

As a first ever attempt at tackling C++ in UE4, I opted to expand my knowledge of steering behaviors.
After already having created a 2D flocking behavior using blended steering,
I now aspired creating formations inspired by the Total War series.

Starting from the top down template provided by UE4, everything is 100% C++.
Blueprints are then derived from the base C++ class in order to get everything working in the engine.


Features:

- Player Controller & Player Pawn
- Actor Spawner
- Formations

Player Controller & Player Pawn

This can be seen as some kind of 'Formation Manager'. It is responsible for moving the units around as a single movement group and for switching between different formations.
Upon recieving input from the player pawn through the player controller, it will calculate a new formation based on: the target location, the current amount of units & the desired formation type.

Actor Spawner

Set up as a child object to the Player Pawn, the Actor Spawner will take care of spawning and destroying units. The spawner will notify the Player Pawn of the creation/destruction of one of its units, making it recalculate the position of the current formation for each unit.

Formations

A formations get calculated from the centerpoint, which corresponds to the location of the Player Pawn.
An array of 2D points is filled with locations corresponding to the chosen geometric figure before being passed on to the Player Controller, who will assign units to each position.
This approach makes it possible to run other algorithms on the array first, before passing it to the units, such as flocking algorithms, or more simpler functions such as assigning each unit to its closest position
