SquirrelMania!

C++ UE4 GroupsWork

Github Link:

Description:

In this course I got through the process used to build games and 3d applications.
The goal was to create a vertical slice of a game, while going through the complete production pipeline from concept to final product.
The following topics got covered: The global production process and the planning of a 3D application. | Design documents. | Version Control Systems. | Team and production manaagement.

Features:

- Parallax Camera:

A parallax effect consisting of 3 layers created through Unreal Shading, Materials & a custom Camera Component.

-Character Controller:

Implementation of a custom Character Controller, together with functionality for multiplayer.
This includes a manager responsible for generating a new Controller and Player Pawn when an input device is connected.

-Combat System:

Functionality for a Charge Attack, Downwards Attack, Normal Attack & Dodge mechanic.
