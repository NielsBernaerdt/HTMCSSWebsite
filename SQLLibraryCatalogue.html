SQL Library Catalogue

C# SQL Library Catalogue .Net Application


Description:

After learning some very basic SQL at school, I wanted to try out if I could manage a simple database using C# and SQL.
In this C# .NET WinForm Application I used a combination of C# and some stored procedures to alter a local database.

Features:

- .Net WinForm Application:

.NET is a widely-used framework in the software development industry. Gaining experience in the framework would undoubtedly benefit me, leaving a positive impact on my development skills.

- SQL Stored Procedures:

SQL Stored Procedures are pre-written, compiled SQL programs stored on a database server. They encapsulate one or more SQL statements into a single executable unit that can easily be executed. This provides improved performance as well as better code organization.

- SQL Server Management Studio:

Microsoft's SQL Server Management Studio is used for the creation of the local database & the created SQL Stored Procedures.
