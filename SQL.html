Description:

*In Progress* Platform Development 4 brings the different skills you learned together and focusses integrating your game in a broader context. This module shows you how to add a (multiuser) backend to your system in order to store, transform and exchange data. In this course you will learn how to use SQL (Structured Query Language). This is a special-purpose programming language designed for managing data held in a relational database management system. Terms like normalization, constraints, mySQL, joins, xml, views, ... will have no secrets for you. You will write SQL-statements as if they are in your mother tongue! At the end of this course, you can create your own database, ready to be used in your future game! Next to basic databases we will learn how to integrate third party platforms as a service in order to allow you to easily set up leaderboard, user analytics and much more.
