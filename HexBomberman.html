HexBomberman

(In Progress)

C++ Graphics Engine using Physx & DirectX11


Description:

In the course of Graphics Programming 2 I learned the finer points of DirectX11, HLSL Shader and PhysX programming.
We got to render 3D scenes in realtime with DirectX 11 and were given a better understanding of the design of commercial and open source game engines.
In this way, we will be able to quickly understand and work with these other game engines.

Features I'm working on:

-Hexagonal Grid:

Instead of the classic Bomberman, this version will be using a hexagonal grid.

-PhysX physics:

Implementation of the physics part of a 3D application including: collision detection & Character Controller.

-HLSL Shaders:

eg: Vertex Skinning shaders for animations, Normal mapping, post processing shaders, ...

-Graphics Engine:

Completion of a custom graphical 3D API and Physics engine.
